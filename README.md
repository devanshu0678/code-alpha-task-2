

A modern and responsive web-based music player built using HTML, CSS, and JavaScript. The player features a beautiful animated UI, playlist management, synchronized lyrics display, sleep timer, theme switching, audio visualizer, and keyboard shortcuts.

## ✨ Features

* 🎶 Audio Playback Controls (Play, Pause, Next, Previous)
* 💿 Animated Rotating Album Art
* 📊 Real-Time Audio Visualizer
* 📝 Lyrics Display with Auto Highlighting
* 🎵 Playlist Management
* 🔀 Shuffle Mode
* 🔁 Repeat Mode
* 🔊 Volume Control & Mute
* ⏰ Sleep Timer
* 🌙 Dark / ☀️ Light Theme
* ✨ Animated Background Particles
* ⌨️ Keyboard Shortcuts
* 📱 Mobile Responsive Design

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎮 Keyboard Shortcuts

| Key   | Action           |
| ----- | ---------------- |
| Space | Play / Pause     |
| ←     | Previous Track   |
| →     | Next Track       |
| ↑     | Volume Up        |
| ↓     | Volume Down      |
| S     | Toggle Shuffle   |
| R     | Toggle Repeat    |
| L     | Show/Hide Lyrics |
| M     | Mute Audio       |
